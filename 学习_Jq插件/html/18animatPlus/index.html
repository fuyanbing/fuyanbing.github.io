AnimatPlus动画插件   Motio   jAni

Motio是一款基于jQuery框架的动画插件它看似简单但功能十分强大，能够创建类精灵化的炫目动画。Motio插件获取一个元素并改变其背景的位置，然后通过Javascript代码来创建动画效果。从本质上来说，Motio插件没有依赖性，但有一个Motio jQuery Plugin版本可供选择使用。
  • 自己移动的背景云层
  • 鼠标经过云层跟着鼠标方向移动
  • 360度全景图片
  • Sprite小精灵
  • 迷你小游戏
jAni是jQuery的一个简单的插件，可以让你动画背景图片。该插件基本上是动画GIF的替代，但具有几个好处。首先，使用动画GIF始终更好，因为所有浏览器都支持这种格式，没有任何JavaScript代码或附加标记，但“黑暗面”则是动画GIF只允许256种颜色，您无法控制动画任何方式.jAni加载一个长度的垂直图像，并以您设置的速度更改其背景位置，从而更好地控制动画。完全轻量级的js代码库。 可以轻松整合到网页应用中。 支持完全用户自定义的CSS样式设定。 支持全部主流浏览器。
