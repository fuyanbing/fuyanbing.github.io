文档

如果你正在读这篇文章, 你可能只是下载 jQuery.mmenu-plugin渴望了解更多吗.
如果是这样的话, 这些演示页面不能更多的帮助. 它们包括例子,颜色主题和一个演示如何实现它与jQuery移动。

这个页面有mmenu-widescreen。css文件包括在内。因此,如果你的屏幕比900px宽,菜单会被打开。
如果没有,点击菜单按钮的标题打开菜单。


调试

当插件试图解决一个问题与HTML或一个选项,调试信息记录到一个公共方法。显示这些信息,覆盖这个公共方法

$.fn.mmenu.debug = function( msg ) {
    alert( msg );
};
